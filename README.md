# 佛陀 · Buddha Skill

> 「善哉，善哉。汝今谛听，当为汝说。」

一个基于大乘汉语佛典的 Claude 对话技能（`.skill`），让现代人得以与大乘语境中的佛陀直接对话，就心灵困惑寻求开示。

本项目完全开源，欢迎有缘人共同充实语料库、改善角色定义。

---

## 功能特点

- 🪷 **角色真实**：言语风格直接取自原典开示句（「善哉」「汝今谛听」「为汝分说」等），非现代心理咨询或 AI 客服腔
- 📖 **语料驱动**：内置核心引文库（《金刚经》《维摩诘经》《圆觉经》等），并支持用户上传 XML TEI / 纯文本佛典进行实时引用
- 🌐 **可扩展架构**：预留梵文、巴利文引文接口；支持按主题扩展专项语料包
- 🤝 **社区共建**：语料库、引文库、对话范式均向贡献者开放

---

## 安装与使用

### 安装

1. 前往 [Releases](../../releases) 下载最新的 `佛陀.skill` 文件
2. 在 [Claude.ai](https://claude.ai) 中：**Settings → Claude's skills → Install skill** → 上传文件
3. 安装完成后，在对话中提出心灵困惑，或直接说「请佛陀开示」，技能即会启动

### 配合语料库使用（推荐）

在 Claude.ai 中建立一个 **Project**（项目）：
1. 左侧栏 → **New Project**
2. Project Settings → **Add content** → 上传本仓库 `corpus/` 中的经文文件
3. 在该 Project 内对话，佛陀将自动从你上传的文本中引用原典

### 直接上传经文

对话时直接将经文文件拖入对话框，佛陀会识别经名、译者，并从文本中取用语句回应。

## 语料来源说明

本项目使用的汉语佛典语料主要来源于：

- **CBETA（中华电子佛典协会）**：[cbeta.org](https://cbeta.org)，提供 XML TEI 格式
  - 授权：学术及非商业使用，须注明来源
- **SAT 大正新修大藏经文字数据库**：[21dzk.l.u-tokyo.ac.jp](https://21dzk.l.u-tokyo.ac.jp/SAT/)

梵文、巴利文语料来源见 `corpus/梵巴原典/README.md`（待建立）。

## 推荐语料清单（按优先级）

### 第一级：必须有
《金刚般若波罗蜜经》（鸠摩罗什译）、《维摩诘所说经》（鸠摩罗什译）、《圆觉经》（佛陀多罗译）、《楞严经》（般剌蜜帝译）、《妙法莲华经》（鸠摩罗什译）

### 第二级：强烈建议
《解深密经》（玄奘译）、《大般涅槃经》选段（昙无谶译）、《胜鬘师子吼经》（求那跋陀罗译）、《无量寿经》（康僧铠译）、《药师琉璃光如来本愿功德经》（玄奘译）

### 第三级：主题补充
《地藏菩萨本愿经》、《般若心经》、《华严经·入法界品》
